# Discord HMS Bot

Discord bot to manage Home Media Servers

## Installation

```bash
npm install
```

## Configuration

```bash
cp .env.example .env
vim .env

DISCORD_TOKEN=YOUR_DISCORD_TOKEN
DISCORD_CLIENT_ID=YOUR_DISCORD_CLIENT_ID
DISCORD_CLIENT_SECRET=YOUR_DISCORD_CLIENT_SECRET
PORT=YOUR_PORT
```

## Usage

```bash
npm start
```
